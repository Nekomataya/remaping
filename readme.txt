「りまぴん（名称変更予定）」ver1.5.8

りまぴん　をダウンロードしていただきましてどうもありがとうございます。
りまぴん は、XPS型式のアニメーションタイムシートを入力・編集するJavascript+DHTML アプリケーションです。

インターネットのサーバ上のアプリケーションをWEBブラウザで実行する他に
ローカルファイルに保存してブラウザでの実行、AdobeAIRアプリケーション
AdobeCS（Photoshop及びAfterEffects）の機能拡張での実行が可能です。

AdobeAIR・Adobe機能拡張での実行にはそれぞれの環境へインストールが必要です。
