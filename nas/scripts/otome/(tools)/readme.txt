readme.txt	nas-Tools

このフォルダは nas(nekomataya animation system)ライブラリの　作業ツールフォルダです。
各ファルダには開発単位ごとのリソースやスクリプトが、入っております。
各フォルダの内容はそれぞれのフォルダの readmeファイルをご覧ください。


autoBuilder	プロジェクト自動構築実験フォルダです
cont	絵コンテ画像を分解してライカリールやビデオコンテに利用する素材を切り出すためのツールです
exe	外部プログラム登録用フォルダです。登録のやり方のサンプルになります。
myScript nasライブラリを使用したＧＵＩスクリプトの簡単なチュートリアルです。
titles	作品別に特化したスクリプトの置き場です。サンプルとしてご覧ください。
utils	nasシステムユーティリティーです。システム設定にご使用ください。
